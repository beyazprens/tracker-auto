## EN

This repository aggregates publicly available BitTorrent tracker URLs.
No tracker is operated, hosted, or controlled by this project.
This project is for informational purposes only.

### CDN Access

You can access the tracker lists via fast public CDNs:

* **jsDelivr (GitHub CDN)**

  ```
  https://cdn.jsdelivr.net/gh/beyazprens/tracker-auto@main/best.txt
  ```

* **GitHub Raw (fallback)**

  ```
  https://raw.githubusercontent.com/beyazprens/tracker-auto/main/best.txt
  ```

These links always point to the latest version on the `main` branch.

## TR

Bu depo, kamuya açık BitTorrent izleyici URL'lerini bir araya getirir.
Bu proje tarafından hiçbir izleyici işletilmez, barındırılmaz veya kontrol edilmez.
Bu proje yalnızca bilgilendirme amaçlıdır.

### CDN Erişimi

Tracker listelerine hızlı ve herkese açık CDN bağlantıları üzerinden erişebilirsiniz:

* **jsDelivr (GitHub CDN)**

  ```
  https://cdn.jsdelivr.net/gh/beyazprens/tracker-auto@main/best.txt
  ```

* **GitHub Raw (yedek / fallback)**

  ```
  https://raw.githubusercontent.com/beyazprens/tracker-auto/main/best.txt
  ```

Bu bağlantılar `main` dalındaki en güncel sürümü gösterir.
